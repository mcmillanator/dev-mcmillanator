A Few Accomplishments

AmSched

AmSched is an amortization schedule manager. An amortization schedule is the math used to calculate the interest on loans. Banks and real-estate companies use it on mortgages. There are other amortization schedule calculators on the net. AmSched is the first that allows you to save many schedules under one account so that you can come back to them later without having enter the loan terms a second time.

This project is not open source. I have gotten good feed back from many people and inted to develop this project into a business.

Darth Vader TODO List

A TODO list for the space nerd in us all. Darth Vader will be pleased that his TODO list does not require a server. JavaScript, HTML, and CSS are all one needs to attack rebel scum. Using JavaScript to update the DOM this page never reloads.

Surf and Paddle

Surf and Paddle is a website I created from a png given to me. I used CSS3 and HTML5 to make a responsive website. With Sinatra I added a dynamic elements so that a single template can be used to create any number of unique pages.

Super Blog

Super blog is a stunning blog design which is responsive and user friendly. Using CSS3, and HTML5 to create a split screen web page that transforms into a stacked web page on mobile devices.

Trackin Dollaz

A balance sheet for us all. Using React, Rails, HTML5, CSS3 my team-mates and I were able to build this balance sheet app. Once an account is created users can save their monthly income and bills. The monthly cash flow is instantly calculated and saved to the database without refreshing the page.

HexMex

A single page app using JavaScript, JSON, HTML5, and CSS3. This app parses a JSON file to create color palettes. Using JavaScript pushState the web page does not refresh when links are clicked or the browser's back button is used. JavaScript can be used in this way to reduce server and bandwidth load by updating only the relevant information instead of the entire DOM.
